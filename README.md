# REPP Artwork (rep_image)

This repository stores a stylized SVG recreation of the REPP artwork.

Files included:
- `rep_image.svg` — stylized neon SVG recreation of the image.
- `README.md` — this file.
- `LICENSE` — MIT license.
- `.gitignore` — common ignores.

Usage
- View the SVG directly on GitHub by opening `rep_image.svg`.
- To embed the raw image in Markdown or websites, use the raw URL after you push the repo:
  - Blob (GitHub view): `https://github.com/<YOUR_USER>/image-hosting/blob/main/rep_image.svg`
  - Raw (direct file): `https://raw.githubusercontent.com/<YOUR_USER>/image-hosting/main/rep_image.svg`
